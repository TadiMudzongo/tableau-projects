Chicago Violence Reduction Dashboard | Calculation Documentation

This page provides technical documentation for key calculations used in the Tableau dashboard, including details on logic, assumptions, and formula design.

1. Weekly Homicide Rate

This metric estimates the average number of homicides per week for a given view context. The formula is:

$$ \text{Weekly Homicide Rate} = \frac{\text{Count of Homicide Victimizations}}{\dfrac{\text{MAX Date} - \text{MIN Date}}{7}} $$

The denominator is calculated as the number of days between the earliest and latest dates in the filtered context, divided by 7 to convert it to weeks. To ensure consistent comparison across all communities and formatting groups, the calculation uses Tableau's Level of Detail (LOD) expression:

$$ \text{Denominator} = \frac{\text{DATEDIFF(day, MIN(Date), MAX(Date))}}{7} $$

This value is computed using: 


SUM((
  DATEDIFF('day',
    {EXCLUDE [Community Area], [Formatting Shapefile Coloumn] : MIN([Date])},
    {EXCLUDE [Community Area], [Formatting Shapefile Coloumn] : MAX([Date])}
  ) / 7.0
))

The numerator is simply the count of homicide victimizations in the current filter context.

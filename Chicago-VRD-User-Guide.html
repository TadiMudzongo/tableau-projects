Chicago Violence Reduction Dashboard | User Guide

This one page guide is designed to help you explore, analyze and interpret the data in this dashboard. Each section provides practical insights on how to use filters, interpret charts, and interact with spatial and time-based views. Whether you're a first-time visitor or a returning analyst, this guide will help you get the most value from the dashboard.

Overview Dashboard

The Overview Dashboard provides a high-level summary of the city’s gun violence landscape, helping you quickly identify which communities are most affected, and how patterns shift across time.

You can explore the trends in the data by making use of the relative date selections in the date filter (e.g., “last 3 months” or “this year to date”). You can also select a crime type using the filter to see how it affects the data displayed.

You can click on any area on the map of Chicago to filter the headline stats as well as the trend of victimizations.

Incident Heatmap Dashboard

This dashboard provides a location-based view of individual victimizations across Chicago. Each point on the map represents a single victimization. You can hover on any point to get precise details on that specific incident.

Zooming in on the map gives you a clearer view of the exact spots where incidents took place, including street-level context from the satellite layer. If you want a larger view, use the full-screen button (highlighted in the screenshot) to expand the map and make navigation easier.

Using Map Layers

Select a map layer to overlay incidents on either community areas or gang territories, helping you explore patterns from different perspectives.

To add context to the incidents shown on the map, try selecting the gang territory layer. This lets you view incidents in relation to known gang boundaries. You can also click on individual gangs to focus the map on incidents around their areas.

Using Page Controls

This dashboard includes page controls, a Tableau feature that lets you step through the data over time using a set of playback-style controls.

To begin using them, set the "Enable Page Controls" option to Enabled. Once enabled, you’ll see controls that let you flip through pages based on a time unit you select.

You can choose to flip by years, months, specific days, seasons, or combined month-year formats. Adjust the speed or pause playback to examine patterns more closely.
